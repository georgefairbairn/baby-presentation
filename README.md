# Let's Make a Baby!

A reveal.js presentation covering everything you need to know about starting a family - pregnancy, birth, the first year, and finances.

**Live site:** https://georgefairbairn.github.io/baby-presentation/

## Running locally

```bash
npm install
open index.html
```
